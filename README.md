# Baofeng 5R Mini BLE Tool

No-build browser tool for reading, editing, and writing Baofeng 5R Mini radio memory over Web Bluetooth.

Open `index.html` through GitHub Pages, then use Chrome or Edge with Web Bluetooth.

The repository intentionally does not include radio memory dumps, read-back files, APK work files, or private channel exports. Generated dumps are ignored by `.gitignore`.

CLI support is in `5r-mini-tool/README-cli.md`.
